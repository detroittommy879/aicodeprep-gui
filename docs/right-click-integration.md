# Right-Click Integration

If you want the smoothest workflow, install the OS file explorer integration.

Then you do not need to open a terminal first.

## What it gives you

It lets you open AICodePrep directly from a folder.

That is useful when:

- you are browsing projects in Explorer, Finder, or a Linux file manager
- you want to jump straight into context prep
- you want this to feel like a normal desktop tool, not a terminal-only utility

## Platform behavior

### Windows

After installing the integration, you can right-click in a folder in Explorer, or right-click a folder itself, and open AICodePrep directly.

### macOS

After installing the integration from the app, use the Finder action on a folder.

### Linux

After installing the integration from the app, use the file manager action on a folder.

## How users install it

The app exposes menu actions to install the integration.

Practical user flow:

1. Run the app once.
2. Open the File menu.
3. Use the install explorer integration option for the current OS.
4. Approve any OS prompts if needed.

## Why this matters

The faster it is to open the app, the more often people actually use it.

That matters because the whole value of AICodePrep is speed and convenience:

open folder -> generate context -> paste into AI chat

## If The Menu Does Not Appear

Try closing and reopening the file manager first.

On Windows, you may also need to restart Explorer or open a fresh Explorer window. On macOS and Linux, the exact behavior depends on the file manager integration installed by the app.
