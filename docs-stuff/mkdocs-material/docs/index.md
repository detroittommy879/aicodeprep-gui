## What this site covers

This site documents what the app can do right now.

It focuses on the real, shipped feature set in the repository today:

- cross-platform desktop terminal behavior
- preset-command workflows and the separate SSH dock
- tabs, splits, and grid layouts
- themes, gradients, fonts, and per-zone typography
- effects, profiles, status indicators, and settings behavior
- current limitations and documentation scope

If you just want a quick overview, start with [What Is w111erd?](/guide/what-is-w111erd). If you want to build and publish this documentation site to your own server, jump to [Building This Docs Site](/guide/building-this-docs-site).
