# What Is w111erd?

w111erd is a cross-platform desktop terminal app for people who want a real shell without committing to a purely keyboard-driven, config-file-heavy experience.

The app still runs real terminal sessions underneath. What changes is the surrounding experience: more visible controls, more adjustable presentation, and a stronger emphasis on readability and personality.

In practical terms, that means:

- preset buttons can run common commands for you
- a separate SSH preset dock can hold connection shortcuts
- tabs, splits, and grid layouts are visible and clickable
- themes, gradients, and fonts are first-class settings instead of hidden tweaks
- profiles let you keep separate setups for different kinds of work
- status indicators make renderer and appearance state easier to understand

## What makes it different

The main idea is not to replace the terminal. It is to make a terminal easier to live with.

That means the app leans into things ordinary people actually notice:

- a button is easier to remember than a shortcut
- a named tab is easier to revisit than an unnamed shell
- a readable font matters more than a pile of hidden config files
- themes, gradients, and layout controls should be visible in settings
- the terminal can feel expressive without stopping being functional

## What it can do today

Today, the app includes:

- normal shell tabs
- horizontal and vertical pane splits
- grid layouts that can show several tabs at once
- preset command buttons and a separate SSH preset dock
- a theme browser with practical, stylized, novelty, chaotic, bundled, and Gogh-derived presets
- theme apply sections for app chrome, terminal colors, fonts, gradients, and effects
- terminal gradients, ANSI palette controls, and separate UI versus terminal font controls
- visual overlays like static and noise
- profile-aware configuration and saved settings
- status bar indicators for renderer, GPU state, links, gradients, and debug mode

## What is still unfinished

Some parts of the UI are groundwork rather than a finished user feature.

Right now:

- the AI Help panel is an early real feature that can send live
  OpenAI-compatible requests, but it still needs stronger secret storage and
  better provider discovery
- older analog TV and CRT displacement effects have been removed from the current build
