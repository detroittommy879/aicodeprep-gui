# Effects and Appearance

## Visual overlays

w111erd supports a small set of optional visual effects that sit over or behind the terminal.

The current effect set is:

- Static overlay
- TV noise
- Simple noise

These are intended as style controls, not productivity requirements.

## Static overlay

![Screenshot Placeholder: Static Overlay Effect](../img/static-overlay-placeholder.png)

The static effect simulates a screen grain or dusty analog texture.

You can tune values like:

- intensity
- density
- amplitude
- opacity
- brightness

## TV noise

![Screenshot Placeholder: TV Noise Effect](../img/tv-noise-placeholder.png)

TV noise adds a more analog, old-screen feel. It is useful if you want the terminal to look rougher or more atmospheric.

## Simple noise

![Screenshot Placeholder: Simple Noise Effect](../img/simple-noise-placeholder.png)

Simple noise is a lighter procedural effect with its own controls for amount, resolution, and optional frame rate.

## Layout density

The app also includes appearance controls that are not strictly effects, such as panel density and layout spacing. These let you keep the interface tighter or airier depending on how much information you want visible at once.

## Removed effects

Some older visual ideas are no longer part of the current build. Analog TV and CRT displacement effects are not user-facing settings now, so they are not part of the supported feature list for this version of the app.
