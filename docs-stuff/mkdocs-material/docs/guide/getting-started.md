# Getting Started

## What you need

w111erd is a desktop app built for Windows, macOS, and Linux.

Once the app is open, you are looking at a real terminal session with an interface wrapped around it.

## The first things to notice

When you first open the app, the important areas are:

- the tab bar across the top
- the preset buttons for quick commands
- the main terminal pane
- the status bar at the bottom
- the settings dialog

## A simple first-run workflow

If you are new to the app, this is the fastest way to get comfortable:

1. Open a new tab from the top bar.
2. Double-click a tab name to rename it.
3. Click one of the preset buttons instead of typing a command from memory.
4. Split the current workspace so you can see more than one terminal at once.
5. Open Settings and try a different built-in theme.
6. Change the terminal font to something more comfortable.

## Fresh machine shortcut

If you are setting up a brand-new machine for AI-assisted coding, open the
`Cool Stuff` menu and choose `Install AI Coding tools/apps`.

That dialog:

- auto-detects the current platform
- explains what the bundled installer script does
- shows the command that will be typed
- opens a fresh tab with the command prefilled so you can review it before you
  run it

## On Windows

If Windows Subsystem for Linux is available, the top preset bar can show a `WSL` button so you can jump into a Linux shell quickly.

## What settings do to the app

Most settings apply live while the dialog is open. That is useful because you can see changes immediately, but it also means the dialog behaves more like a live control panel than a traditional form.

If you change something and want to back out, use the dialog's revert action before closing.
